# Breast_cancer_detection
Breast cancer detection using histopathology images

How to predict?
Run on terminal: python new_predict.py -j infer.json

new_predict.py: python code to make a prediction
normal.png: Normal image in png format
cancer.png: Cancer image in png format
test_jap.py: Fastai code for training using 2 gpus
test_jap2.py: Faastai code for training and inference using 3 gpus
