# Breast_cancer_detection
Breast cancer detection using histopathology images

# How to predict?
Run on terminal: python new_predict.py -j infer.json

# Description
1. new_predict.py: python code to make a prediction

2. normal.png: Normal image in png format

3. cancer.png: Cancer image in png format

4. test_jap.py: Fastai code for training using 2 gpus

5. test_jap2.py: Faastai code for training and inference using 3 gpus
